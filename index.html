Course Specification

Course Title: Data Visualization Design

Course Code: HCI 3401

Credit Hours: 4 Hours

Level/Year: 3rd Year / 5th Level

Pre-requisites: HCI2203 - User Research, DS2201 - Introduction to Database

Course Description & Objectives

This course focuses on principles and techniques of designing effective data visualizations within the context of Human-Computer Interaction (HCI). Students learn to represent complex data in intuitive ways using design principles and cognitive psychology.

Main Objectives:

  • Understand foundational principles of visual perception.
  • Design user-centered, accessible visualizations.
  • Utilize modern tools for interactive visual representations.
  • Critically evaluate effectiveness and ethical considerations.
